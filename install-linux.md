# Install HyperFrames Skills on Linux

Verify Node:

node -v
npm -v

Install HyperFrames skills:

npx skills add heygen-com/hyperframes

Verify installation:

ls ~/.agents/skills

Expected folders:

hyperframes
hyperframes-cli
gsap
website-to-hyperframes
hyperframes-media
hyperframes-registry
lottie
three
tailwind
