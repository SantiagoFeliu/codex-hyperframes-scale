# Install HyperFrames Skills in Codex

Open terminal:

cd ~/your-project

Install HyperFrames skills:

npx skills add heygen-com/hyperframes

Verify installation:

ls ~/.agents/skills

Expected folders:

hyperframes
hyperframes-cli
gsap
website-to-hyperframes
hyperframes-media
hyperframes-registry
lottie
three
tailwind

Restart Codex if skills do not appear immediately.
